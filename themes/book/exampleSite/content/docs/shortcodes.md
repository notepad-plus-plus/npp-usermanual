# Shortcodes

## Expand shortcode

### Default

```tpl
{{</* expand */>}}
## Markdown content
Lorem markdownum insigne...
{{</* /expand */>}}
```

{{< /expand >}}

```tpl
{{</* expand "Custom Label" "..." */>}}
```
